Note Taking WebApp.
Made in React.js (HTML, CSS, JS)

Users can take notes, Delete notes and toggle between Dark and light mode as per preference.
Will soon have a edit note functionality.

## Features:
- Add new notes
- Delete notes
- Responsive design
- Built with modern react practices

## Tech Stack
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation
```bash
npm install
npm run dev
